2. Notes

System requirement：

Android 6.0 or above.
4G or more memory.
200M storage on SD card.

The webserver environment:

Nginx:1.10.1
Php:5.6.27
MySql:5.5.21
Samba:3.5.22

The default MySQL administrator:

User name:root
Password:cbf

Important notice：

  • The network storage sharing (Samba) function needs the rooted android system. You can modify the sharing name, user name and password here. The cellphone needs to reboot if you modified the sharing name.
  • After the Root the webserver could use the 80 and 443 ports.
  • After the Root the webserver will not be terminate by the battery management module of the system when the screen is off or the Barknet exits.
  • If you would like to change the SSL certification keys，please put the server.key, server.csr and server.crt to the [SD card]/ExpertWeb/php/sbin folder and restart the web server。
  • If you want to use sepcial icons，you can just put the icons into the [SD card]/ExpertWeb/icons directory to use it.
